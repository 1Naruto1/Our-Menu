# 电子厨房 GitHub 版 V2

这是一个纯前端、纯本地保存的电子厨房应用，适合部署到 GitHub Pages 后在 iPad / iPhone Safari 中使用。

## 功能

- 菜品菜单：添加菜名、图片、适合餐次、食材与用量、标签和备注
- 冰箱食材：添加食材图片、余量、单位、保质期和备注
- 食材提醒：自动显示 3 天内要过期的食材
- 今日推荐：根据冰箱食材和快过期食材推荐菜品
- 三餐打卡：早餐 / 午餐 / 晚餐打卡，记录做饭人、评分和评论
- 本地备份：支持导出和导入 JSON 备份

## GitHub Pages 上传

把本文件夹中的所有文件上传到 GitHub 仓库根目录：

- `index.html`
- `app.js`
- `styles.css`
- `manifest.webmanifest`
- `service-worker.js`
- `assets/` 整个文件夹

如果仓库里有旧版同名文件，直接覆盖即可。不要只上传某几个文件，否则页面可能还是旧缓存或缺少图片。

## 手机 / iPad 更新后看不到新版

Safari 可能会缓存旧版。更新后可以尝试：

1. 等 1-3 分钟再刷新 GitHub Pages 链接。
2. 在 Safari 里长按刷新，或关闭页面重新打开。
3. 如果已经添加到主屏幕，先打开网页确认新版，再重新添加到主屏幕。
4. 必要时导出备份后清理 Safari 网站数据，再导入备份。

所有菜品、食材、图片和打卡记录都保存在当前浏览器本地 IndexedDB 中，不会自动同步到其他设备。
